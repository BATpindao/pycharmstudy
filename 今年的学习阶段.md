# Python学习路线：从零到独立完成电商AI私人任务

目标：掌握Pandas数据处理 + 训练/微调大模型，实现商品标题自动分类 + 电商智能客服  
总时长：约6-12个月（每天2-4小时）

## 阶段1：Python 基础（3-4周）
核心内容：变量、数据类型、控制流、函数、文件读写、异常处理、基础OOP

学习资源：
- 视频：B站“黑马程序员 Python教程”（全套免费）
- 视频：B站“莫烦Python”（基础部分）
- 书籍：《Python编程：从入门到实践》（第3版）
- 官方：Python官网教程（docs.python.org/zh-cn/3/tutorial/）

阶段结束项目：命令行个人记账本工具
- 需求：CSV存储记录，支持添加/查看/按类别统计/余额计算，带错误处理
- 效果：可独立运行的交互式记账工具，数据持久化，代码150-200行

## 阶段2：数据处理核心库（Pandas + NumPy）（3-5周）
核心内容：DataFrame操作、read_excel/csv、清洗、groupby、merge、可视化

学习资源：
- 视频：B站“宋天龙 Pandas数据处理”
- 视频：B站“黑马程序员 Pandas教程”
- 官方：Kaggle “Pandas”课程（免费互动）
- 书籍：《Python for Data Analysis》（第3版）

阶段结束项目：电商商品Excel数据清洗与统计仪表盘
- 需求：自动清洗、添加价格区间、类目统计、生成图表、输出新Excel
- 效果：一键处理原始电商数据，得到清洗版+带图表的报告Excel

## 阶段3：机器学习基础（scikit-learn）（4-6周）
核心内容：特征工程、文本向量化、分类算法、Pipeline、评估

学习资源：
- 视频：B站“吴恩达 机器学习”（中文字幕）
- 视频：B站“黑马程序员 机器学习”
- 官方：scikit-learn官网文档
- 书籍：《机器学习实战》或《机器学习》（西瓜书）

阶段结束项目：商品标题文本分类器（传统ML版）
- 需求：Tfidf+随机森林，支持交互预测和Excel批量分类
- 效果：准确率80%+，能批量处理标题Excel并导出结果

## 阶段4：深度学习与PyTorch基础（6-8周）
核心内容：Tensor、nn.Module、DataLoader、训练循环、简单CNN/LSTM

学习资源：
- 视频：B站“莫烦PyTorch”（中文动画讲解）
- 视频：B站“吃瓜教程 PyTorch”
- 官方：PyTorch官网教程（pytorch.org/tutorials）
- 课程：fast.ai Practical Deep Learning

阶段结束项目：基于PyTorch的商品标题深度文本分类器
- 需求：实现LSTM/CNN，支持分词、训练曲线、批量预测
- 效果：准确率高于传统ML，支持交互和Excel批量分类

## 阶段5：大语言模型使用与微调（6-10周）
核心内容：Transformers、pipeline、LoRA微调、RAG、Gradio

学习资源：
- 官方课程：Hugging Face 免费课程（huggingface.co/learn）
- 视频：B站“LangChain中文教程” + “LoRA微调教学”
- 项目：GitHub “LLaMA-Factory”（一键微调）
- 文档：Transformers官方文档

阶段结束项目：微调后的电商智能客服原型
- 需求：LoRA微调中文LLM + RAG（上传FAQ Excel），Gradio网页界面
- 效果：支持上传知识库的实时智能问答，多轮对话

## 阶段6：项目整合与部署（4-8周）
核心内容：完整项目流程、Streamlit/Gradio部署、模型优化

学习资源：
- 视频：B站“Streamlit教程” + “Gradio中文教程”
- 官方：Streamlit文档 + Gradio文档
- 实战：Hugging Face Spaces部署示例

阶段结束项目：完整电商AI工具套件
- 需求：Streamlit多页面应用，整合标题分类 + 智能客服
- 效果：本地Web应用，实现全部私人任务，可在线部署

坚持完成，你将拥有独立开发AI工具的能力！加油！
